# Rever Cafe

Rever Cafe is a concept-driven digital experience inspired by the ambiance of a greenhouse-style cafe.  
It blends cinematic storytelling with immersive design, creating a space where light, flavor, and atmosphere converge.

## ✨ Features
- Cinematic hero sections with layered video backgrounds  
- Elegant typography and gradient overlays for depth  
- Interactive elements with smooth motion transitions  
- Responsive layouts for seamless viewing across devices  
- Brand identity highlights: artisan coffee, curated events, exclusive pastries  

## 🎯 Purpose
This project was created as a **portfolio showcase** to demonstrate skills in creative frontend development, motion design, and storytelling UI/UX.  
It is **not a paid project** and **not a promotional collaboration** — purely a personal design and development exercise.

## 🌿 Vision
Rever Café is designed as a digital canvas where every detail feels intentional, inviting users to savor the experience.  
It reflects a passion for blending **design, motion, and narrative** into polished user-centered interfaces.
